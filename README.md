# Customer Churn Analysis & Prediction Project

## Project Overview

This project implements an end-to-end **Customer Churn Analytics &
Prediction System** using: - SQL Server (ETL & Data Modeling) - Power BI
(Dashboards & Reporting) - Python + Machine Learning (Random Forest)

The goal is to analyze customer behavior, understand churn drivers, and
predict future churners for proactive marketing actions.

------------------------------------------------------------------------

## Business Objectives

-   Analyze customer data by **Demographic, Geographic, Account, Payment
    & Services**
-   Study churner profiles and churn reasons
-   Build churn KPIs (Churn Rate, New Joiners, Total Churn)
-   Predict future churners using Machine Learning

------------------------------------------------------------------------

## Tech Stack

-   SQL Server & SSMS
-   Power BI
-   Python (Pandas, NumPy, Scikit-learn)
-   Jupyter Notebook
-   Anaconda

------------------------------------------------------------------------

## Project Architecture

CSV → SQL Server (Staging) → SQL Server (Production) → Power BI\
                                                     ↘ Excel → ML Model

------------------------------------------------------------------------

## Key Metrics

-   Total Customers\
-   New Joiners\
-   Total Churn\
-   Churn Rate

------------------------------------------------------------------------

## Machine Learning

-   Algorithm: **Random Forest Classifier**
-   Target Variable: Customer_Status (Churned / Stayed)
-   Output: Predicted churn customers for campaign targeting
